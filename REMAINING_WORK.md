# Remaining Work

下面是这个整合版目前还缺少、但如果要继续往“更像学习助手”推进时最值得补的内容。

## 1. 内容理解层

- OCR：现在桌面模式主要还是直接看截图，没有可靠文本抽取
- 版面分析：还不能稳定区分标题、正文、图表、公式、代码块
- 公式识别：数学公式和表格仍然是明显短板

## 2. 学习助手能力

- 更细的任务模板：术语讲解、图表分析、习题思路、复习闪卡
- 追问上下文：目前只有很轻量的最近上下文，没有真正的连续问答
- 笔记导出：还没有一键保存学习卡片或 Markdown 笔记

## 3. 桌面模式体验

- 区域选择：现在默认抓主屏幕，还不能只抓某个窗口或选区
- 主窗口预览：还没有显示最近截图缩略图
- 手动触发“深入分析”：目前以自动检测变化为主

## 4. 视频模式体验

- 更细的进度反馈：现在主要是日志文本
- 中断恢复：长视频中途停止后不能断点续跑
- Windows 字体适配：注释视频渲染仍沿用了 lecture-lens 的跨平台 fallback 逻辑

## 5. 模型与运行环境

- GPU 环境自动检查：目前只会在模型加载时报错或警告
- 本地模型管理：还没有模型目录检测、下载提示和配置持久化
- 错误恢复：网络失败、模型权限失败、FFmpeg 缺失时仍偏原型级

## 6. 如果只做课后项目，最建议优先补这三项

1. OCR + 页面文本抽取
2. 只截取当前窗口/选区
3. 导出分析结果为 Markdown 笔记
